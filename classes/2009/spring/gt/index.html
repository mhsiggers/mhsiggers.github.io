Syllabus

In the contents of the text, there are sections that are starred. We will cover these sections. This will include an introduction to graphs, matchings, Menger's Theorem, planar graphs, graph colourings, network flows, extremal graph theory, Ramsey theory, random graphs, and the graph minor theorem for trees.
