Syllabus

We will spend about a week (one class) on each of the following topics. They are listing in the order they occur in the text, but we may mix this up a bit.
  • Graph Basics (2 classes)
  • Tree search algorithms
  • Networks, Max-flow min-cut algorithm
  • Computational complexity and graph homomorphisms
  • Connectivity and Menger's Theorem
  • Planarity and Kuratowski's Theorem
  • Extremal Graph Theory: Ramsey and Turan Theorems
  • Probabilistic Method (* if time): Evolution of Random Graphs, Erdos' graph incomparbility proof.
  • Vertex colouring, Brooks' Theorem
  • 4-colour problem, List colouring, Thomassen's Five-list-colouring theorem.
  • Matchings, Hall's Theorem
  • Edge Colouring, Vizing's Theorem
  • Hamilton Cycles, Dirac's Theorem
