Syllabus

In this course we introduce discrete structures and counting techniques, and use them to introduce formal written proofs. We introduce symbolic logic and set theory to give validity to basic proof techniques such as proof by contradiction. We see recursive definitions and inductive proofs. We use graph problems to introduce proof by minimal counter-examples.

Computational techniques are invesigated, but the emphasis is on proof techniques.
