Syllabus

This is a first class in Algebra. We will follow the text, seeing basic group theory, in which we will learn about such things as subgroups, cosets, isomorphisms, homomorphisms, factor groups and products so that we can state without proof the structure theory of finite abelian groups. We will finish of by looking at rings and fields.

This is a proof heavy course. You should have taken some class, like set theory, number theory, or discrete mathemathics, in which you have written proofs before.
