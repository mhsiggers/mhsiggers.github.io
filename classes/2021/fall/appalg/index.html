Syllabus

This class covers the same material as the Cryptography (암호론) class that was last taught in spring 2019. We will cover substitution ciphers, discrete logarithms, Diffie-Hellman Key exchange, ElGamal and RSA and other public key cryptosystems, probabilistic encryption, the Vigenere cypher, elliptic curve cryptography, digital signatures and hash functions. To do this we will have to touch on the following background topics: computational complexity, modular arithmetic, finite fields, quadratic residues, probability, and elliptic curves.
