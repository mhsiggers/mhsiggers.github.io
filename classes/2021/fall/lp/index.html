Syllabus

We expect to all of "Understanding and Using Linear Programming". Here are the chapter titles, we can expect to spenc 1-2 weeks on each of these.
  • Introduction
  • Examples of Problems that we will solve
  • Integer Programming and LP relaxation
  • Theory of Linear Programming
  • Simplex Method
  • Duality
  • Beyond the Simplex method

Grading

Students will be expected to present topics several times over the semester. These presentations will be graded, and count for 50% of the mark. Assigned problems will count for the rest of the mark.
